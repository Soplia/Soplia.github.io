HPC Server
  1. I can not access the Linux server brehat and yoshi by direct access (ssh).
  2. 连接到Linux系统服务器ssh jrxie@totoro以及SLURM计算节点网关ssh jrxie@frontalhpc
  3. Slurm 是一个开源、容错、高可伸缩的集群管理和大型小型 Linux集群作业调度系统。slurm不需要对操作系统内核进行修改，而是相对独立的。
  4. 不管登录 frontalhpc 还是 totoro，家目录，是透明的，即从这两个服务器上看到的都是相同的。
  5. Can not use scp on Mac neither pscp on Windows at home.
    • 不在Limos时，需要首先连接 ext.limos.fr 然后在其中再连接 totoro等，所以如果要上传文件，是否需要先上传到ext.limos,fr 然后再在其中上传到totoro等呢？
    • 我进行了尝试，结果 公钥失效了 😳
    • 嗯嗯，我的猜想是正确的，因为使用VPN后就能够上传文件了
VPN 链接
  1. 到Limos指定页下载并安装连接VPN，全程无难点
    • 在连接时端口号为22，其余均不用更改
  2. 上传，一定要注意不要使用 ~ ， 或者 ~/ 要如下一样给出正确的文件夹位置
    pscp -r ./00-jiaruixie/code jrxie@totoro:/home/etud/jrxie/insp
  3. 下载
    pscp jrxie@totoro:/home/etud/jrxie/insp/code/01-ecm/ecm.m ./00-jiaruixie
  4. 可以看出不管是上传还是下载，都是在本地进行的；当然从server上应该也是可以的，但是没有进行测试；scp 、 pscp的使用没有什么困难的，直接参照网上的教程即可

How to use VPN in ISIMA
  1. go to https://doc.isima.fr/vpn/installation/ try to install the VPN
  2. go to https://doc.isima.fr/vpn/connection/ configure the connection with the username of the university
  3. use ssh connect to berzet
    • ssh berzet -l jrxie
    • exit
  4. After use the vpn, just disconnect.
