Python

基础

  1. python最具特色的就是使用缩进来表示代码块，不需要使用大括号

  2. 每个变量在使用前都必须赋值，变量赋值以后该变量才会被创建。

  3. 两个整型对象 1 和 2 的分配给变量 a 和 b，字符串对象 “runoob” 分配给变量 c

  4. 字典中，值可以取任何数据类型，但键必须是不可变的，如字符串，数字或元组。

  5. 字典中想要删除某一个键值对，直接删除该键值对的键就行了

  6. Python

  7. 在 Python 中，使用了 yield 的函数被称为生成器（generator）

  8. 调用一个生成器函数，返回的是一个迭代器对象
  9. a, b = b, a + b该语句是首先同一完成右边表达式的计算之后，再进行统一赋值
  10. 加了星号 * 的参数会以元组(tuple)的形式导入，加了两个星号 ** 的参数会以字典的形式导入
  11. 想使用 Python 源文件（模块），只需在另一个源文件里执行 import 语句
  12. Python解释器就依次从这些目录（搜索路径sys.path）中去寻找所引入的模块
  13. 模块除了方法定义，还可以包括可执行的代码。这些代码一般用来初始化这个模块。这些代码只有在第一次被导入时才会被执行。
  14. 一个特别的模块 sys ，它内置在每一个 Python 解析器中。变量 sys.ps1 和 sys.ps2 定义了主提示符和副提示符所对应的字符串
  15. 导入语句遵循如下规则：如果包定义文件 init.py 存在一个叫做 all 的列表变量，那么在使用 from package import * 的时候就把这个列表中的所有名字作为包内容导入
  16. 当处理一个文件对象时, 使用 with 关键字是非常好的方式
  17. 当内部作用域想修改外部作用域的变量时，就要用到global和nonlocal关键字了

Python简介

  • python语法简洁清晰，特色之一是强制用空白符作为语句缩进。
  • Python具有非常丰富和强大的库。被称为胶水语言，能够把用其它语言制作的各种模块很轻松地连接在一起。
  • Python是完全面向对象的语言。
  • Python采用动态类型系统。在编译的时候，Python不会检查对象是否拥有被调用的方法或者属性，而是直至运行时，才做出检查。所以操作对象时可能会抛出异常。不过，虽然Python采用动态类型系统，它同时也是强类型的。
  1. 导入模块
    • import math
    • from math import *
    • form math import pi
  2. 在编写Python的时候尽量不要使用Tab键进行空格。
  3. 如果使用文本编辑器编辑Python的时候需要及时进行保存。
  4. 推荐的图书：《Introduction to the Theory of Computation》
  5. 在进行递归程序编程的时候不应该顺着程序进行递归调用，应该假设我已经获得了n-1的阶乘，能否通过现有的算法获得n的阶乘。
  6. Python中对于实例的属性的定义绝对是个很奇葩的方式。
  7. 使用 numpy.correlate求解相关系数的时候，输入参数是两列，然后输出参数是2$\times$2的矩阵（因为是两列）。具体请看Numpy.cov Example
    • 如果函数看不懂，那么首先需要去查找这个函数，然后确定形参与返回值。
